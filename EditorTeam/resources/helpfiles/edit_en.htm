"Search text":

Allows you to find the specified fragment in the text. Selected in the context menu or by pressing "Ctrl+F".

"Copy text format":

Copies the selected text format to the clipboard.

"Apply text format":

Applies the text format specified in "Copy text format" to the selection.

"Align text right":

Aligns the selected piece of text to the right. Selected in the context menu or by pressing "Ctrl+K".

"Align text left":

Aligns the selected piece of text to the left. Selected in the context menu or by pressing "Ctrl+L".

"Align text center":

Aligns the selected piece of text to the center. Selected in the context menu or by pressing "Ctrl+M".

"Switch font":

Changes the font size of the selection.
