"New":

A new file is created. Called in the context menu or by pressing the "Ctrl+N" keys.

"Open":

The file selection menu opens. Called in the context menu or by pressing the "Ctrl+O" keys.

"Open file read only":

The function opens the file as read-only. Called from the context menu or by pressing "Ctrl+R".

"Close":

When this function is selected, it is possible to save changes, when confirming actions, or cancel changes and exit without saving. Called in the context menu or when pressing the "Esc" keys.

"Save":

The menu for saving the edited file opens. Called in the context menu or by pressing "Ctrl+S".

"Save as":

The menu for saving the edited file opens, in which you can select the extension of the saved file (For example: name.txt, name.tha, etc.). Called in the context menu or when pressing "Ctrl+Alt+S" keys.

"Export to PDF":

Exports the entered text to *.pdf format and saves it in the specified file. Called in the context menu or by pressing the "Ctrl + T" keys.

"Print":

Sends the entered text to be printed. Called in the context menu or by pressing the "Ctrl + P" keys.

"Exit":

Exit the editor program. If text is entered, a choice is offered: save or exit without saving. Called in the context menu or when pressing "Ctrl+Q" keys.
